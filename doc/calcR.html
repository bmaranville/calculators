Loading data:

The loader is expecting tab or space-delimited columnar data files, with the columns in the order Q, R, dR ... (more columns are permitted but ignored). The dR should be a 1-sigma error bar on the reflectivity. Header lines prepended with # will be ignored. For polarized data, a single file with the desired cross-sections concatenated with two carriage returns between them (\n\n) is expected, where the polarization is indicated in a header line before the data in each section as e.g. (this is the format exported by the NCNR online reduction system) 
# "polarization": "++"
0.01 0.99 0.01
...

Calculations

The reflectivity is calculated using a slab model with Nevot-Croce roughness as in ref. [1]
The magnetic calculations including Zeeman effects are described in [2].

Export to Refl1D

In order to get a full fit including instrumental resolution, parameter correlation estimates and true statistial error bars on the model, the model prepared here can be exported to the powerful Refl1D program by simply pushing the "generate script..." button. This script can be directly imported to Refl1D, and includes a reference to any data files loaded.
